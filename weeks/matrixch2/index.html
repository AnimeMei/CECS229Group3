Def n - vector

Let F\mathbb{F}F be a field.
An n-vector is an element of the form (x1,x2,...,xn)∈F×F×F×...×F=Fn(x_1, x_2, ..., x_n) \in \mathbb{F} \times \mathbb{F} \times \mathbb{F} \times... \times \mathbb{F} = \mathbb{F}^n(x​1​​,x​2​​,...,x​n​​)∈F×F×F×...×F=F​n​​

Def D-vector

Let D be a finite set, and F\mathbb{F}F be a field. A D-vector over F\mathbb{F}F is a function from D to F\mathbb{F}F.

F.g. The fuction f: WORDS →R\rightarrow \mathbb{R}→R is a D-vector RWORDS\mathbb{R}^{WORDS}R​WORDS​​
